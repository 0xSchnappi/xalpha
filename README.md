xalpha

[![doc](https://readthedocs.org/projects/xalpha/badge/?style=flat)](https://xalpha.readthedocs.io/)

关于国内基金投资的全流程管理。

场外基金的信息获取，投资记录与整合分析，以及简单的策略回测。

## Documentation

文档地址： https://xalpha.readthedocs.io/. 

## Installation

```bash
pip install xalpha
```

目前仅支持 python 3 。
